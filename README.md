# House-price-prediction
House Price Prediction ML project using Python and Linear Regression to estimate property prices based on area and bedrooms.

# House Price Prediction ML Project

## Description
This project predicts house prices based on area and number of bedrooms using Linear Regression.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Algorithm
Linear Regression

## How to Run
pip install -r requirements.txt  
python house_price.py
